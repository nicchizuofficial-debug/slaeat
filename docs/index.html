SLAEAT プライバシーポリシー

最終更新日：2026年6月21日

1. 収集する情報

位置情報

周辺の飲食店を検索するため、アプリ使用中のみ現在地（緯度・経度）を取得します。位置情報はホットペッパーグルメAPIへの検索リクエストに使用され、サーバーへの保存や第三者への提供は行いません。

お気に入りデータ

お気に入り登録した店舗情報（店舗名・ジャンル・住所・URL）はお使いの端末内にのみ保存されます。外部サーバーへの送信は行いません。

2. 使用しない情報

  • 氏名・メールアドレス・電話番号などの個人情報
  • 広告識別子（IDFA / GAID）
  • クラッシュレポートや分析データ

3. 第三者サービス

本アプリはホットペッパーグルメAPI（株式会社リクルート）を利用して飲食店情報を取得します。位置情報はこのAPIに送信されます。
リクルートのプライバシーポリシー：https://www.recruit.co.jp/privacy/

4. お子様のプライバシー

本アプリは13歳未満のお子様を対象としておらず、意図的にお子様の個人情報を収集することはありません。

Privacy Policy (English)

SLAEAT uses your device location (latitude/longitude) only while the app is in use to search for nearby restaurants via the HotPepper Gourmet API. Location data is not stored on any server or shared with third parties beyond this API call. Favorite shop data is stored locally on your device only. We do not collect any personal information, advertising identifiers, or analytics data.
